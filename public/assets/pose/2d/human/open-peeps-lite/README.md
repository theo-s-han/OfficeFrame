# Open Peeps Lite (Segmented)

- Source: [Open Peeps](https://openpeeps.com/)
- Original author: Pablo Stanley
- Original license: CC0
- This folder contains a project-modified segmented SVG pack made after studying official Open Peeps reference samples.
- Purpose: first pose-ready external-style pack for `/pose` 2D human preview.

Notes:

1. This is not the original full Open Peeps library.
2. These parts were simplified and rearranged to work with the local joint-based pose renderer.
3. Future packs can replace or extend this folder with more accurate part separation.
